# Roadmap

This roadmap describes possible directions for CoverFix.

## Current focus

- Fix missing, blank, or broken Apple Music playlist covers on macOS
- Keep the project as a simple Python tool
- Keep the web UI local and lightweight
- Avoid turning CoverFix into a general music manager
- Focus on playlist artwork, not album artwork metadata

## Short-term goals

- Keep README and usage instructions accurate
- Improve safer preview and dry-run behavior
- Improve error messages for macOS permission failures
- Confirm behavior across supported macOS versions
- Add more terminal examples and screenshots

## Medium-term goals

- Detect more playlist cover failure cases
- Improve restore and backup workflows
- Add export options before modification
- Make the Apple catalog fallback more transparent and configurable
- Improve compatibility with iCloud Music Library sync delays

## Long-term ideas

- Optional automatic playlist cover generation
- Better batch repair workflow for large libraries
- More detailed diagnostics for Apple Music cache and sync issues
- Native macOS app only if there is clear demand and a maintainable release path

## Not planned for now

- General MP3/M4A/FLAC tag editing
- Album artwork repair inside audio files
- Full Apple Music library management
- Cloud music service
- Web account system
- Mac App Store release
